Python Installation Guide¶

Regardless of how you choose to install Python, please make sure you install Python version 3.x. The latest increment of version 3 is ideal. Python 2 is no longer supported. As part of your educational experience you will use jupyter notebooks, a programming environment that runs in a web browser throughout class. For this to work you will need a reasonably up-to-date browser. The current versions of the Chrome, Safari and Firefox. For more information see the browsers supported by Jupyter.

We recommend the use of the Anaconda distribution to maintain your Python environment for most data science applications. Python is currently the most popular data science language used in industry. A major reason that it is an industry standard today is the diverse and well-maintained ecosystem of packages. Anaconda is essentially a bundle of many of these packages and it comes with Conda, a package management system.

Note

You do not need administrator or root permissions to install Anaconda if you select a user-writable install location.

Download the image or install script from https://www.anaconda.com/download/

On Ubuntu¶

  1. install via the command line

~$ bash ~/Downloads/Anaconda3-2020.02-Linux-x86_64.sh

The above link may not have the same name as the actual file downloaded

Important

Answer yes to questions that you are prompted with

  1. Restart terminal

On OSX¶

  1. Run the installer on the downloaded image

Important

Answer yes to the path adding question

  1. restart terminal

On Windows¶

  1. Ensure that you install the Anaconda installer for Windows for Python 3

  2. Double-click the .exe file

  3. Follow the instructions on the screen

If you are unsure about any setting, accept the defaults. You can change them later. When installation is finished, from the Start menu, open the Anaconda Prompt.

Important

Ensure that you check Make Anaconda the default Python

https://conda.io/docs/user-guide/install/windows.html

Keeping conda up-to-date¶

~$ conda update --all

Troubleshooting¶

Removing an old version¶

~$ rm -rf ~/anaconda*
